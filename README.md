# Teste-cypress-do-site-El-shammah-crew

Testes automatizados E2E com Cypress desenvolvidos no site do grupo de dança El-Shammah Crew. O objetivo foi não apenas divulgar o site, mas também demonstrar na prática as habilidades em automação de testes logo no início da jornada como QA.

---

## Motivação

O site do El-Shammah Crew foi criado por mim, então fazia todo sentido utilizá-lo como base para praticar automação de testes. Além de validar o funcionamento do site, o projeto serviu como estudo prático de Cypress no início da minha atuação como QA Engineer.

---

## Cenários Automatizados

- Verificação do banner rotativo nos 3 slides
- Navegação pelo menu e validação de cada seção
- Verificação dos integrantes do grupo no carrossel
- Verificação dos momentos históricos do grupo (25+ eventos)
- Validação dos links e redes sociais do rodapé
- Verificação dos botões de contato com `target="_blank"`
- Validação das redes sociais de cada integrante
- Validação dos links de cada momento histórico

---

## Tecnologias

- Cypress
- JavaScript
- cypress-file-upload (plugin para upload de arquivos)

---

## Pré-requisitos

- Node.js instalado
- npm instalado

---

## Instalação

```bash
npm install
```

## Executar os testes

**Modo headless:**
```bash
npm test
```

**Modo interativo:**
```bash
npm run cy:open
```
